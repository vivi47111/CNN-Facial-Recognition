# CNN Facial Recognition & Feature Map Dimensionality Analysis

## 1. Project Overview
This project focuses on the systematic design and analysis of a **Convolutional Neural Network (CNN)** for facial recognition using the **Yale Face Dataset**. Beyond simple classification, the study delves into the internal mechanics of the network by tracing high-dimensional tensor transformations across layers.

## 2. Key Research Highlights
* **Dimensionality Derivation**: Provided rigorous mathematical derivations for feature map sizes, ensuring the structural integrity of the deep architecture.
* **Feature Activation Analysis**: Evaluated the density of feature activations to interpret how the model differentiates between various facial identities.
* **Performance Tuning**: Balanced computational efficiency and top-1 accuracy by optimizing kernel sizes and pooling strategies.

## 3. Technical Implementation
* **Model Layers**: Integration of Convolutional layers, **Batch Normalization**, and **ReLU** for stable convergence.
* **Optimization**: Utilized **Adam** optimizer to minimize cross-entropy loss.
* **Visualization**: Generated montage views of internal feature maps to validate the "locality" of learned representations.
